# 🛡️ PhishLens

> “Not just detecting phishing… but understanding it.”

## 📌 About

PhishLens is an AI-powered phishing detection and awareness platform built with a simple idea:
people shouldn’t just be told *something is dangerous* — they should understand **why**.

I started this project while learning cybersecurity and AI, driven by curiosity about how phishing actually works and how people get trapped by it.

Instead of being just another detection tool, PhishLens focuses on **explainability and learning** — making users smarter, not just safer.

---

## 🚀 What I’m Building

* 🔍 Detect phishing using AI (BERT for text & URLs)
* 🖼️ Analyze fake webpages using Vision Transformers
* 🧠 Explain *why* something is phishing (SHAP / LIME)
* 🌐 Chrome extension for real-time detection
* 💬 Interactive chatbot to simulate phishing scenarios

---

## 🧠 Tech Stack (Learning + Building)

* Python (FastAPI)
* React (Chrome Extension)
* HuggingFace Transformers
* SHAP / LIME
